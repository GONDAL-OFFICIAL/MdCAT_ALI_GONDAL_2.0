## ⚙️ Run Locally

**Prerequisites:**

- Node.js (v18 or later)
- npm (v9 or later)

**Steps:**

```bash
# 1️⃣ Clone the repository!
git clone https://github.com/GONDAL-OFFICIAL/MdCAT_ALI_GONDAL_2.0.git
cd MdCAT_ALI_GONDAL_2.0

# 2️⃣ Install dependencies!
npm install

# 3️⃣ Start the development server!
npm run dev
```
